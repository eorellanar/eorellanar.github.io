Project Overview

Proposal

The website goal is to inform users about the basics and principles of JavaScript. We will go over datatypes, variables, functions, and conditional statements just to name a few. This website is meant for anyone who is first hearing about JavaScript and wants to learn more about it. Within our pages you'll find the basics of how to get started, coding examples, and some intermediate concepts about JavaScript.

Site Map

Page Design

  • Home page
    • An introduction for the rest of the site and the concepts it will hold
    • Intended for anyone showing an interest and JavaScript and wondering what kind of information the site will hold
    • Will include basic text about what the page is about and images to go along with it. No to very little JavaScript will be used here
    • For reduced rendundency, all pages (inluding this one) will have a navigation bar in the header to navigate between pages.
  • About Us
    • Includes information about the developer and client
    • Intended for people looking to find more information about the developer and client
    • Include text and photos relating to the developer and client. Will have basic JavaScript for dropdown menus to show/hide information.
  • Basics of JavaScript
    • Includes information about basic concepts of JavaScript
    • Intended for those wanting to learn about JavaScript and its basics
    • Will include text, images, slideshows, and other imageries. Will have JavaScript to illustrate coding examples and information. Buttons will be used showcase JavaScript code
  • Higher Leveled Concepts
    • Covers concepts and logic beyond that basics. Includes more complicated code
    • Intended for an audience that wants to dive deeper in JavaScript and learn how to write more complicated code.
    • Similar to the basics page, will include text and imagery about higher level concepts. JavaScript will be used to illustrate and showcase information. This includes hiding and showing information/images and having buttons to showcase JavaScript code.
  • Example Projects
    • This will be a showcase page of JavaScript Projects
    • Intended for users who want to see real examples of JavaScript being used
    • Will include integrated projects on the webage. Will have buttons to activite functions and methods. Will also have buttons that showcase pieces of code and inform the user on what they do. A list project will also be used to gather user info and display it.
