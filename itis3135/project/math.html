Operators and Arithmetic in JavaScript

Below we'll be talking about how to do math in JavaScript

Arithmetic Operators

JavaScript supports basic math operators for performing calculations: 

    let a = 10;
    let b = 3;
    
    console.log(a + b); // Addition → 13
    console.log(a - b); // Subtraction → 7
    console.log(a * b); // Multiplication → 30
    console.log(a / b); // Division → 3.333...
    console.log(a % b); // Modulus (remainder) → 1

Assignment Operators

These are used to assign values to variables, often with arithmetic: 

    let x = 5;
    x += 2; // x = x + 2 → 7
    x -= 1; // x = x - 1 → 6
    x *= 3; // x = x * 3 → 18
    x /= 2; // x = x / 2 → 9
    x %= 4; // x = x % 4 → 1

Comparison Operators

Used to compare values and return a Boolean (true or false): 

    console.log(5 == '5');   // true (loose equality)
    console.log(5 === '5');  // false (strict equality)
    console.log(10 != 5);    // true
    console.log(10 !== '10'); // true
    console.log(7 > 3);      // true
    console.log(2 <= 1);     // false

Logical Operators

Combine or invert Boolean values: 

    let isAdult = true;
    let hasID = false;
    
    console.log(isAdult && hasID); // false (AND)
    console.log(isAdult || hasID); // true (OR)
    console.log(!isAdult);         // false (NOT)

Increment and Decrement

Handy for quickly increasing or decreasing numbers by 1: 

    let count = 0;
    count++; // count = 1
    count--; // count = 0
